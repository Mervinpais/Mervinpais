Whats a 'Mental Crash'? MentalCrash is a(n) (Eso) Programming language i made to try and make a language which we can code in one line, ONLY

Although, we can write our program in multiple lines

Nerd Ack: "Uhm Acktually that defets dhe whol purpos of yur program becuz u shoud maik it soo it ONLY one lien ruhns"

But that would be impractical because it would be;

  1. Hard to read
  2. Not easy to maintain

Here is how you would write a Hello world! program in MentalCrash

MentalCrash

p Hello World!

This is not that hard;

  • First, we split the first space, so 'p' and 'Hello World!'

  • Second, we go through each list of command and their argumentment, so here, 'p' is the (only) command (here), and 'Hello World!' is the (only) argument (here)

  • Third, We do whatever command needs to do with whatever argument/parameter it has, here p uses the first parameter, since its the first command, 'p' is eqivalent to 'print()' in python, or 'Console.WriteLine();' in C#.

  • Finally, then go on with the other commands till the end, then after the program will turn into a shell, unless there is a '.' command, and at the '.' command, we stop the program, even with other commands after it

Wow, its not *that* hard
